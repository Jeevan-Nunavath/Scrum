# Scrum

Resume Match Assistant - Scrum Portfolio Project

This repository documents my Agile/Scrum portfolio project for a Resume Match Assistant concept. The idea is to help job seekers compare their resume or profile against a target job role, identify improvement areas, and organize follow-up actions.

The main focus of this project is project coordination, not full software development. It demonstrates how I planned the project using Jira and Confluence, including epics, sprint backlog items, story point estimates, resource roles, QA planning, sprint retrospectives, risks, and final project closeout.
